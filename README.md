Usage
---------

Apply middleware to sg-server, which is internally use by sugo-cloud.

```javascript
/** This is an example to use sugo-middleware-noop */

'use strict'

const sgServer = require('sg-server')

const server = sgServer({
  middlewares: [
    require('sugo-middleware-noop')({
      // Options
    })
  ],
  routes: {
    /* ... */
  }
})

server.listen(3000)


```
